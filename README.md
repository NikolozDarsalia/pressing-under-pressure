# pressing-under-pressure
Spatial analysis of Bayer Leverkusen's pressing behavior in the 2023/24 Bundesliga season.  Uses vector geometry (sf), raster operations (terra), and a Difference-in-Differences design  to show that Leverkusen pressed anomalously high when losing late — benchmarked against  2015/16 top Bundesliga teams. BSE Geospatial Data Science term paper.
